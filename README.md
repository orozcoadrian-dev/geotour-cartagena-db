Database Project Repository

This repository contains all SQL scripts and development phases required to build and deploy the database for the classroom project from scratch.

Includes
- Database creation scripts
- Table definitions
- Primary and foreign key constraints
- Relational structure
- Initial data insertion scripts
- Query testing scripts
- Database development phases
- Structured documentation for implementation and maintenance

Purpose
⭐​ | The goal of this repository is to centralize and organize the complete database logic, making development, version control, collaboration, and future scalability easier for the project team.
